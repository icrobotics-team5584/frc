Automatic config.json file to RoboRIo bat files for windows using scp.

Step 1 (if not already done ONCE)
Create a desktop shortcut for the following two bat file. 
"setUpRioCon".bat
"configToRIO".bat

Step 2 (ALWAYS)
Connect to the robot

Step 3 (do only ONCE repeat if necessary or if using different robot)
Run "setUpRioCon".bat (by double clicking the shortcut) 
Type “yes” when prompt and press enter

Step 4 (send the config file from your computer to RoboRio)
Run "configToRIO".bat  (by double clicking the shortcut)
